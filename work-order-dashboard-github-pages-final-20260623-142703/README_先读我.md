# 工单五模块看板 Codex 开发包

本包用于让 Codex 基于 4 类工单数据，继续开发“飞书多维表格 + 网页看板”的联动版本。

## 已包含文件

- sample_data/：用户上传的 4 个原始 CSV
- normalized_data/工单统一明细_260618.csv：已统一后的工单明细，增加了“工单类型、处理人员、未结案天数、风险等级”等字段
- normalized_data/项目客诉专项跟进模板.csv：项目类客诉专用跟进字段
- docs/01_Codex一次性指令.md：建议直接复制给 Codex 的完整指令
- docs/02_业务需求说明.md：业务需求说明
- docs/03_字段口径与规则.md：字段映射、风险规则、五大模块说明
- docs/04_飞书多维表格设计.md：飞书表结构与仪表盘建议
- prototype/工单五模块看板原型.html：可直接打开的本地网页原型

## 当前规则

- 工单类型：质量工单、支持工单、市场工单、供应工单
- 处理人员：优先取“制单人”；支持工单无制单人时取“创建人/CC起草人”
- 结案判断：最终结案、工程师结案、已结案、闭环回访等视为结案
- 高风险：未结案天数 > 6
- 中风险：未结案天数 > 4 且 <= 6
- 低风险：未结案天数 <= 4
- 模块一：所有类型工单信息总汇总，支持下探查看具体进度和卡点
- 模块二到五：分别为质量工单、支持工单、市场工单、供应工单
